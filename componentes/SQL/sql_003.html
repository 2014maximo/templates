CONSULTAS SQL SERVER DESDE DBEAVER

En esta ocasión se realizan procesos de consulta a una base de datos SQL Server, basados en una extructura ya diseñada:

northwind.sql

Se utiliza el administrador de base de datos DBEAVER, en vez del comunmente utilizado SQL SERVER MANAGEMENT.

SELECT * from orders

Esta consulta trae todo el contenido de la tabla "ORDERS".

SELECT orderid, customerid, shipcity
FROM orders

Esta consulta trae los "ORDERID", "CUSTOMERID" Y "SHIPCITY" de la tabla "ORDERS".

SELECT * FROM customers
WHERE companyname like 'B%'

Esta consulta trae todo de la tabla "CUSTOMERS" donde "COMPANYNAME" tenga la letra "B" al comienzo.

SELECT * FROM Customers
WHERE CustomerID LIKE 'Blaus'

Esta consulta trae todo de la tabla "CUSTOMERS" donde "CUSTOMERID" tenga "Blaus" en el contenido.

SELECT * FROM Customers
WHERE CustomerID LIKE 'B%'
ORDER BY ContactName

Esta consulta la tabla "CUSTOMERS" mientras "CUSTOMERID" tenga la letra "B" al inicio y ordenados por el atributo "CONTACTNAME".

SELECT * FROM Customers
WHERE CustomerID LIKE 'B%'
ORDER BY ContactName DESC

Esta consulta la tabla "CUSTOMERS" mientras "CUSTOMERID" tenga la letra "B" al inicio y ordenados por el atributo "CONTACTNAME" en orden descendiente.
