ELEMENTOS Y DESCRIPCIÓN

SQL

Lenguaje de Consulta Estructurado. SQL es un lenguaje de programación para administrar las bases de datos relacionales, posee comandos referentes a DDL(Definición) Y DML(Manipulación).

Sistema de gestión de bases de datos

(SGBD) Se refiere a todo el grupo de programas que permiten almacenar, modificar y extraer los datos. Es decir toda la interfaz que permiten al usuario ingresar a la información de manera organizada.

Dato

Cualquier valor que represente un número, letra, símbolo. Que en su concepto simple no representa una información completa o de importancia relevante.

Base de datos

El lugar donde están almacenados los datos de manera organizada, sincronizada y que permitan su fácil acceso.

Interfaz DB

Es el entorno desde donde un usuario interactúa con la información de la base de datos.

DDL Lenguaje de definición de datos

Es la parte en el lenguaje de base de datos donde se crean las estruturas que van a contener los datos, sin referirse a la manipulación de los datos.

DML Lenguaje de maniputación de datos

Es la parte en el lenguaje de base de datos donde se manipula los datos almacenados.

DBA Administrador de bases de datos

El responsable administrador de bases de datos.

Componentes de una base de datos

Practicamente en cualquier base de datos, existen 4 elementos primordiales: tablas, consultas, informes y formularios. Todos son indispensables y necesarios:

Las tablas permitirán almacenar los datos.

Con las consultas podremos acceder a los datos almacenados, ordenarlos y filtrarlos por diferentes criterios.

Los informes serán la forma más eficaz de presentar nuestros datos.

Con los formularios se facilitan las tareas de introducción de datos.
