PROCEDIMIENTO EN WINDOWS

Una vez descargadas las herramientas el procedimiento es fácil. Se instala el node js, de manera normal sin marcar opciones avanzadas, luego se agrega el comando para instalar "angular CLI" usando una consola de comandos, y ya lo último es instalar el editor de código preferido.

COMANDOS CON "ANGULAR CLI" ÚTILES

npm install -g @angular/cli

1. Con este comando se instala "Angular CLI" y todas las opciones para Angular.

ng new nombre_aplicacion

2. Crear un proyecto Angular nuevo para empezar a desarrollar.

ng serve

3. Compila el proyecto y lo expone en una url para visualizarlo en un navegador.

ng serve o

4. Lo mismo que el comando anterior pero abre el proyecto automáticamente en el navegador.

ng serve --port 4201

5. Compila el proyecto en un puerto específico, en este caso saldría por el "localhost:4201"

ng component ruta/nombredelcomponente

6. Crea un conjunto de 3 tipos de archivos: nombredelcomponente.html, nombredelcomponente.ts, nombredelcomponente.spec.ts

ng g c nombre/componente --skipTests

7. Lo mismo que lo anterior sin el archivo de pruebas.spec.ts

ng g s ruta/nombreArchivoService.ts

8. Crea un archivo.service.ts con la extructura de un archivo de servicios de Angular base para desarrollar.

ng update @angular/cli @angular/core

9. Actualiza la version de "Angular CLI".
