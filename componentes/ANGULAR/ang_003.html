INPUT Y OUTPUT

Los decoradores Input y Output son excelentes elementos de angular para intercambiar datos entre componentes, ya sea entre componentes que hagan de padres o de hijos, incluso entre hermanos.

Input

El decorador Input es más simple de implementar y de entender cómo funciona. Mensionando tambien que permite pasar datos de un componente que contiene a otro de esta manera:

Hijo

Mediante @Input() el componente hijo recibe los datos a través de la vista html, vista anteriormente.

Padre

Desde el componente padre la data es simplemente una variable global.
